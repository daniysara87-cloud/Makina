# Makina Web App

App web Next.js para generar un archivo MIDI en navegador con estilo makina / remember.

## Probar en local

```bash
npm install
npm run dev
```

## Publicar en Vercel

1. Sube esta carpeta a GitHub.
2. Entra en Vercel.
3. Importa el repositorio.
4. Pulsa Deploy.

## Archivo principal

- `app/page.tsx`
